    ```
2) [Yuzu](https://yuzu-emu.org/)

    ```
## Sidebar.json
The sidebar.json is the file that holds the data for width for it the y can't be changed as I don't know how to do that just yet
    ```json
    [
        {
            "Sidebar": "200"
        }
    ]
    ```
Change 200 to what ever width you want.
